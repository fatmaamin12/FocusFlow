2. BACKEND REQUIREMENTS
====================================================
- Use Flask as the backend framework.
- Must include at least one Python Standard Library module other than 'random'
  Examples: os, datetime, json, pathlib, re, uuid
- Must contain at least one Python class:

    * Includes methods
    * Uses __init__() to initialize object state
- Must include persistent storage:
    * tasks.json
    * users.json
- Backend must:
    * Load and save JSON data
    * Handle authentication (login, signup, logout)
    * Serve all required pages
    * Provide API endpoints (/get_tasks, /save_tasks)
    * Handle sessions or user state
